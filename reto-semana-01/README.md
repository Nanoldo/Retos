# Reto semana 01

Programa que limpia valores numéricos y los convierte a enteros y los suma.

## Instrucciones de uso

Use un archivo de texto con los números que desea trabajar como entrada y guarde la salida en otro archivo de texto.

```bash
python3 main.py < entrada.txt > salida.txt
```

## Ejemplo de entrada y salida

**Entrada:**

41,

-239251

-318,-490,431,-448,-648,907,-518,7,133,-283,-555,514,615,-462,-824,-770,851

899y$n,3F(22,t4n@85,303&_,zw%472,6[,9D90,UU84,3f5,17K5

**Salida:**

41

-239251

-1858

37711
